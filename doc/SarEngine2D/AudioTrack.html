用于播放音频的独立音轨。

Syntax

var audioTrack = new SarEngine.AudioTrack(audio);

Parameters

SarEngine.Audio audio
一个可播放的SarEngine.Audio音频对象。

Returns

returns

Properties

audio
SarEngine.Audio 在此音轨上播放的音频对象。
audioElement
AudioElement 此音轨播放时所使用的HTML AudioElement
duration
Number 音轨当前播放所经历的时间长度， 单位为秒。
output
Output 该音轨的输出接口。
playing
Boolean 表示该音轨是否正在播放。

Methods

addToOutput(output)
将该音轨添加到输出接口。
end()
结束播放。
pause()
暂停播放
play()
开始播放。
removeFromOutput(output)
将该音轨从指定的输出接口移除。

Events

onEnd
在该音轨完成播放后触发。
